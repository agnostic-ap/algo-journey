## 算法学习记录

这个仓库用于记录算法题解与学习笔记。

- **语言**: JavaScript / TypeScript（可根据需要扩展）

### 目录结构

- `problems/`：按平台与难度存放题目代码
  - `leetcode/easy/`
  - `leetcode/medium/`
  - `leetcode/hard/`
- `notes/`：算法、数据结构、刷题思路整理
  - `data-structures/`：如 `array.md` 等
  - `algorithms/`：如 `binary-search.md` 等
  - `patterns/`：题型模板，如 `sliding-window.md`
- `tools/`：辅助小工具或脚本
